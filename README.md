# 小学管理系统

一个基于Vercel和Supabase的现代化小学管理系统，包含班级管理、学生管理、教师管理和成绩管理等功能。

## 🚀 功能特点

- 📊 **数据看板** - 系统概览和统计数据
- 👥 **班级管理** - 管理1-6年级的班级信息
- 👨‍🎓 **学生管理** - 学生信息录入和管理
- 👩‍🏫 **教师管理** - 教师信息管理
- 📝 **成绩录入** - 批量录入学生成绩
- 📈 **成绩报表** - 成绩统计和分析

## 🛠️ 技术栈

- **前端**: HTML5, CSS3, JavaScript (ES6+)
- **后端**: Supabase (PostgreSQL数据库)
- **部署**: Vercel (全球CDN)
- **样式**: 原生CSS，支持响应式设计
- **图标**: Font Awesome 6
